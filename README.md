## Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/misafxd/shoppingCartTOP.git
   ```
2. Navigate to the project directory:

   ```bash
   cd <shoppingCartTOP>
    ```

3. Install the dependencies:

    ```bash
   npm install
    ```
4. Build the project:

    ```bash
     npm run build
    ```
After completing these steps, the project will be ready to use.
